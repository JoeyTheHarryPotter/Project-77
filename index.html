Project on console

Visual Studio is a taskbar website that you can create JavaScript in. First, go to visualstudio.microsoft.com/downloads/. Next, fill out all of the questions. Then, create a new file in your downloads section. Lastly, File > Open Folder > Chose the new folder you created and you can now create your first project!!!

The console screen is so important becasue it can help you determine if you have an error in the console screen. The console screen is just for testing purposes though. If you want to oepn the console screen, press F12 (this could be different if you a different device).
